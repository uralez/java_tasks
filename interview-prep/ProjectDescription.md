# Interview Prep — лог разработки

## Описание проекта
Веб-приложение для подготовки к техническим интервью.
Студент регистрирует свои ответы на вопросы, получает оценку от оценщика,
отслеживает динамику оценок по темам и вопросам.

**Стек:** Java 23, Spring Boot 3.5.11, Gradle, PostgreSQL 17.5,
Liquibase, Hibernate/JPA, Spring Security 6, Thymeleaf, Lombok

**Пакет:** `com.ayakovlev.interviewprep`

**CSS переменные:** `--bg`, `--surface`, `--border`, `--accent: #3d6b4f`,
`--accent2`, `--text`, `--muted`, `--error`, `--radius: 14px`

### #3 — Redesign answer panel
- Textarea на всю панель, автоподгонка высоты под текст (`autoResizeTextarea`)
- Grade, date, evaluator спрятаны в кнопки-чипы с inline-редакторами:
  - клик вне редактора → применяет значение
  - ✕ → отменяет без сохранения
  - evaluator: клик на имени → применяет и закрывает
- Две верхних панели объединены в одну ЗП (статистика + таблица)
- ПП и СП переделаны в стиль ЦП (только просмотр, без редактирования)
- Три режима раскладки (переключатель в шапке ЦП):
  - Режим 1: ЗП | ЦП
  - Режим 2: ЗП | ПП | ЦП
  - Режим 3: ПП/ЦП сверху, ЗП/СП снизу
- Режим 3 скрыт на мобильных экранах
- Раскладка реализована через CSS `grid-template-areas` + `data-mode`

### #4 — Красивый confirm-dialog вместо стандартного
- Кастомный диалог вместо браузерного `confirm()`
- Брендинг: заголовок "Interview Prep" шрифтом Playfair Display

- Показывает id и дату удаляемого ответа
- Закрытие: ✕, кнопка Отмена, клик вне окна

- Стили в `main.css`
